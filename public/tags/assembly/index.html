Primitive Data Types in Assembly

How to calculate range of data types? These are built-in and predefined data types can be used for declaring variables; Integer Float Double Character Boolean Wide Character Void And they have modifiers like; Short Long Signed Unsigned Each of them will allocate a memory space and have a range of values they can hold related to this space. For example, char requires 1 byte, int requires 4 byte, long and double requires 8 byte.
