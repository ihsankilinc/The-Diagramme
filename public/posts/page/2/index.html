How to convert String to Array in JavaScript?

If you want to convert your string into an array without writing complex code you have 4 options; Using String.split() Using Spread Operator […String] Using Array.from(String) Usign Object.assign([],string) 1. Using split() Method This is the most popular method for converting a string to an array and is also used for a few other javascript tricks like replacing all of the occurrences for given character. It is not a destructive method which means it doesn’t change the original value, just returns another copy of it.
